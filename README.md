# 🦟 Arboviroses Serra/ES — Análise de Doenças Tropicais

Projeto desenvolvido para a disciplina **Projeto Integrador III – Ciência de Dados** da FAESA, curso de Ciência da Computação.

## 📌 Sobre o projeto

Este projeto analisa a **distribuição geográfica e sazonalidade de arboviroses** — Dengue, Zika e Chikungunya — no município de **Serra/ES**, no período de 2015 a 2023.

Utilizando técnicas de ciência de dados, buscamos identificar padrões de incidência por bairro, períodos de pico, e possíveis correlações com fatores climáticos e socioeconômicos, gerando insights que possam apoiar a vigilância epidemiológica local.

---

## 🎯 Problema

Serra é um dos municípios com maior índice de notificação de arboviroses no Espírito Santo. Apesar disso, não existe uma análise pública sistematizada que integre dados de casos, clima e indicadores sociais em nível de bairro. Este projeto preenche essa lacuna.

---

## 🗂️ Fontes de dados

| Fonte | Dados |
|-------|-------|
| DATASUS / SINAN | Notificações de dengue, Zika e Chikungunya |
| IBGE | Dados socioeconômicos por setor censitário |
| Prefeitura de Serra | Shapefile e dados por bairro |
| INMET | Temperatura e precipitação mensais |
